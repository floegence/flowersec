Flowersec Proxy Sandbox Demo (Browser)

This demo shows flowersec-proxy/http1 + flowersec-proxy/ws in runtime mode: a Service Worker intercepts requests for /examples/ts/proxy-sandbox/app/ and forwards them to a Flowersec runtime.

Paste a connect_artifact envelope, or use the dev server to mint an artifact-first proxy bootstrap payload. The demo now reads the proxy.runtime scope embedded in the artifact to configure the service worker instead of re-typing it here.

Latest runtime scope: none
